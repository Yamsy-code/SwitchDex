üîß 3. ADVANCED CONFIGURATION
===========================

3.1 ADMIN ROLE SETUP
--------------------
Set a role that can manage bot settings:

/setadminrole role:@Moderators

Remove admin role:
/removeadminrole

3.2 ROLE MENTIONS
-----------------
Ping specific roles for updates:

/setmentionrole category:homebrew role:@HomebrewTeam
/setmentionrole category:firmware role:@TechTeam

Stop pinging:
/removementionrole category:homebrew

3.3 QUIET HOURS
--------------
Set times when notifications shouldn't be sent:

/setquiethours start:22:00 end:08:00 timezone:EST

Disable quiet hours:
/removequiethours

During quiet hours, notifications are queued and sent when quiet hours end.

3.4 DIGEST MODE
---------------
Receive daily summary instead of instant notifications:

/setdigestmode mode:on time:09:00 timezone:EST

Disable digest mode:
/setdigestmode mode:off

3.5 CUSTOM REPOSITORY TRACKING
-------------------------------
Track specific GitHub repositories:

/addtracking link:https://github.com/developer/repo

You can also specify a custom display name:
/addtracking link:https://github.com/developer/repo name:"My Custom App"

Remove tracking:
/removetracking
